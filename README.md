# Conversor de Medidas
Um conversor de diferentes medidas feito em JavaScript e completamente portado para a web.

## Ferramentas utilizadas
- HTML
- JavaScript

## Conversões disponíveis
- Metro para pé
- Pé para metro
- Quilômetro para milha
- Milha para quilômetro

## Árvore de desenvolvimento
```bash
├── modules         -- pasta com os módulos em js para conversão
│   ├── im
│   │   ├── ft_mt.js
│   │   ├── km_ml.js
│   │   ├── ml_km.js
│   │   └── mt_ft.js
│   └── sm
├── README.md
├── utils
│   └── index.js    -- exporta o diretório "modules" para o main.js             
├── main.js         -- exporta os módulos para o html
└── index.html
```
> _Todos os arquivos inclusos neste repositório até o presente momento se encontram na Primeira Versão_
